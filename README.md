# ⚡ OVERCLOCK

A futuristic cyberpunk-inspired TechFest landing page built using HTML, CSS, and JavaScript.

## 🚀 About

OVERCLOCK is a visually immersive landing page designed for a technology festival. Inspired by futuristic control systems, cyberpunk aesthetics, and high-performance computing, the website delivers a bold and energetic experience through custom animations, interactive sections, and a striking red-and-black visual theme.

The concept revolves around pushing technology beyond its limits — an "overclocked" environment where innovation, creativity, and competition collide.

## ✨ Features

- Cyberpunk-inspired UI/UX
- Animated performance gauge
- Interactive hero section
- Event showcase cards
- Speaker spotlight section
- Festival timeline
- Registration call-to-action
- Fully responsive design
- Custom SVG graphics and animations

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript
- Google Fonts
- SVG Animations

## 🎨 Design Highlights

- Industrial hazard-strip styling
- Futuristic typography
- Dynamic redline theme
- Custom hover animations
- Responsive grid layouts
- System dashboard-inspired visuals

## 📸 Preview

OVERCLOCK simulates a futuristic control panel where users can explore event information, schedules, speakers, and registration details in an immersive techfest environment.

This project was created to practice:
